接口

Kotlin 的接口与 Java 8 类似，既包含抽象方法的声明，也包含实现.
它可以有属性但必须声明为抽象或提供访问器实现.
接口不会保存属性值，实现接口时，必须重写属性.
使用关键字 interface 来定义接口.
