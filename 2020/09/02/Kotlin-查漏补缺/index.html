区分和使用 run, with, let, also, apply

函数 是否是扩展函数 函数参数(this、it) 返回值(调用本身、最后一行)
with 不是 this 最后一行
T.run 是 this 最后一行
T.let 是 it 最后一行
T.also 是 it 调用者本身
T.apply 是 this 调用者本身

使用 T.also 函数交换两个变量

b.also { b = a } 会先将 a 的值 (1) 赋值给 b，此时 b 的值为 1，然后将 b 原始的值（2）赋值给 a，此时 a 的值为 2，实现交换两个变量的目的。
